Request Header Microservice

Documentation

  • The API displays the user's information including: IP address, preferred language and system information
  • The API endpoint is [base URL]/api/whoami and returns JSON data

Example of usage

api/whoami

Example Output (JSON)

{
"ipaddress": "76.116.9.29",
"language": "en-US,en;q=0.5",
"software": "Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:62.0) Gecko/20100101 Firefox/62.0"
}
